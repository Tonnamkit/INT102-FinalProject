เป็ยครั้งแรกที่ได้มาอบรมที่ม. เข้าอบรมเรื่อง “การเขียน JavaScript เพื่อเรียกใช้บริการ Google Map API”  วันนั้นคิดว่ามาเช้าแล้วแต่ก็มีเพื่อนมาเช้ากว่า เพื่อนๆ มาจากต่างจังหวัดกัน เลยมาทำความรู้จักกัน
ทั้งเพื่อนซึ่งได้เพื่อนทั้ง 3 สาขา IT,CS,DSI ครึ่งเช้าจะเป็นการฟังเรื่องเล่าของรุ่นพี่ที่เรียนจบแล้วได้ทำงานกับ Google มาเล่า Road map ชีวิตของเขา ให้คำแนะนำและตอบคำถาม ซึ่งดีมากเป็นการสร้างแรงจูงใจที่ดี
ทำให้เป็นเหมือนแรงผลักดันให้เราอยากทำให้ได้เหมือนกัน จบเซคชั่นนี้แล้วต่อไปจะเป็นการพดถึง API ความหมายคืออะไร เกริ่นคร่าวๆ หมดจบครึ่งวันเช้าพักกลางวันไปกินข้าว แล้วทำความรู้จักกับเพื่อนๆ เพิ่มอีก
ทำให้รู้ว่าเพื่อนๆ ที่มาเรียนที่นี้มีหลายจังหวัด ต่อครึ่งบ่ายจะเป็น Workshop เริ่มเขียนภาษา JavaScript และภาษา HTML เบื้องต้นสำหรับเว็ปที่จะเรียกใช้ Google Map API ก็ได้ความรู้ เมื่อมีคำถามก็สามารถ
